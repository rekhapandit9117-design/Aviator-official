Installation Steps

1

Get APK File

Homepage par jaakar "DOWNLOAD OFFICIAL APK" button par click karein. Aapki file automatic background mein save hona start ho jayegi.

2

Enable Unknown Sources

Agar aapko warning milti hai, toh apne phone ki Settings ➔ Security mein jayein aur "Unknown Sources" ya "Install Unknown Apps" permission ko allow karein.

3

Execute Installation

Apne browser ke Downloads manager ya File Explorer mein jaakar Avaiter_Official.apk par click karein aur "Install" select karein.

4

Launch and Play

Installation finish hone ke baad app icon par click karke application open karein, register karein aur flight parameters track karna start karein!
