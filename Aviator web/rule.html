Official Game Rules

To guarantee a fair, highly transparent, and competitive environment for our growing active community, all players must adhere strictly to the rules of engagement outlined below:

  • Multipliers: Multipliers scale in real-time. Flight outcomes are determined via secure cryptographic baseline variables before take-off.
  • Single Account Policy: Each player is restricted to a single verified profile. Creating multiple accounts to exploit server ecosystems will lead to terminal termination.
  • Third-Party Tools: The utilization of modified APK variations, unauthorized analytical bots, scripts, or latency manipulation software is strictly prohibited.
  • Disconnection Protocol: In case of unexpected connection loss, the algorithm executes actions matching server-side records to protect the user profile balance pool.

Privacy Policy

At AVAITER, protecting your privacy is our primary objective. We build next-generation security boundaries around your gameplay logs and system access metadata.

We handle basic identifier fields like dynamic user handles, device architecture models, and operating system metrics to deliver customized connection patches and scale frame-rates seamlessly. We never sell or rent user profile operational records to any third-party corporate databases.

All communications running across the app dashboard are handled inside a standard 256-bit encrypted data socket layer.

Data Retention Policy

We maintain user account data matrices exclusively to enhance community health and protect the core user leaderboard framework.

  • Active Profile Records: Match statistics, ranking records, and transactional history updates remain accessible throughout the continuous operational lifecycle of your profile.
  • Account Deletion: If you file a formal profile wipe inquiry, all associated verification points and history metrics will be permanently scrubbed from our active servers within 7 working days.
  • System Logs: Server error logs and network telemetry data are automatically deleted after 30 days to preserve storage efficiently.
